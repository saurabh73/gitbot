Hello World!

I am gitbot, I give out basic information about the selected repository in your slack channel, help you fetching your repository code and most importantly commits to your repository with uploaded code in your channel.

Developed using Stdlib

Basic Info

use /info command in slack channel to get basic information like description, stars, forks, etc about the selected repository

Fetch

use /fetch command in slack channel to get selected repository archived and uploaded to the channel

Commit

upload zip, mention @GITBOT with commit message, select your repository and branch to commit your code
